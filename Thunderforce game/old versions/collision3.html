Collision detection exercise, Hannah Dee

A steerable player box

This canvas has some boxes, and you have to write the "collides" function which determines when they overlap

  1. Study the code carefully.
  2. If there are things that don't make sense, just ask.
  3. Have a go at writing the collides function (there are some hints at the bottom).
  4. When you've done that, make the black blocks move: you can do this how you like, one thing you could try is to just change x or y during the game loop, and re-draw them somewhere else when they go off the screen.
  5. When you've done that, try to make something happen when there is a collision (delete a black block and make it appear somewhere new? have an explosion? a prize with bonus points? up to you).
